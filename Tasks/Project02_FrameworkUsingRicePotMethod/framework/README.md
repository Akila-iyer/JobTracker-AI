# API Testing Framework

This framework provides automated tests for ReqRes API using Pytest and Requests.

## Features
- Modular test structure
- Data-driven testing
- Positive and negative test cases
- APIClient abstraction

## Requirements
- Python 3.8+
- Pytest
- Requests

## Usage
1. Install requirements: pip install -r requirements.txt
2. Run tests: pytest framework/tests -v
